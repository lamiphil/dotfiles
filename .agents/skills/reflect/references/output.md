## Response Format

Present any learnings in the specified output format.

```markdown
## Session learnings

**Worth adding to AGENTS.md:**

- [concrete proposed line or bullet, written in the voice/style of the existing file]
  Why: [one line explaining what it prevents or clarifies]

**Worth adding to project documentation:**

- [concrete proposed line or bullet]
  Why: [one line]

**Worth adding to a skill:**

- [skill name and concrete proposed instruction]
  Why: [one line]

**Considered but rejected:**

- [candidate] — [why it did not clear the bar]
```

Then ask: "Want me to apply these edits?". NEVER edit docs or skills without explicit approval.

If nothing clears the bar simply output:

```markdown
## Session learnings

Nothing worth codifying. [One sentence explaining why.]
```
